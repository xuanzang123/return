// 上传

请把整个 return 文件夹上传到你的网站服务器根目录。

// 访问

通过网址 http://[yourdomain]/return/stat_login.php 登录统计系统查看数据，[yourdomain] 注意要替换成你的网站域名。

初始用户名密码都是 “admin”，修改密码请登录后到系统设置里边修改帐号密码

// 退订

return_output.txt 此文件的内容为用户点退订后显示的内容，可以自行修改，支持 HTML 代码。

// IP 数据库

统计 3.5 版本开始支持“纯真 Ip 数据库”。
下载最新版本的“纯真 Ip 数据库”后，把“QQWry.DAT”文件改成“qqwry.dat”放入 return 目录下上传到网站即可直接使用。
注意：Linux 服务器，务必把文件名改成纯小写字母名“qqwry.dat ”
